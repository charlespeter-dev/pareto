CAPSULE login instructions

1. Account Set-Up

The link in the welcome email will bring you to this screen to set up your account.

Email: Please use the email that the invitation was sent to. A security code will be sent to this email on the next screen.

Cell Phone: Please use your cell phone number–a 6 digit security code will be sent to the number you provide.

2. Email Confirmation

An email will be sent with a 6 digit security code to the email that the account was created under.

This email will come from noreply@paretohealth.com, so please check your spam in case it gets sent there. You can also request an additional code using the “Resend Code” button if you do not receive this email.

3. Login Into Your Account

Once you enter your six digit code, the screen will prompt you to login using the password you created when you were setting up your account.

4. Need Help?

If you need help you can always email Customer Support or click the “Need Help” button at the bottom of the login screen.

The “Need Help?” button will prompt you to send an email to the above address so that our IT team can assist you in your account setup.
